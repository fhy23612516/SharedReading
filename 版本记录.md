7. API 冒烟测试增加轻量列表响应、轻量心跳响应和手机进度监听源码校验。

## v0.7.1 - 2026-05-01

6. `package.json` 新增 `npm run reset-password -- <account>` 入口。
7. API 冒烟测试新增 JSON 模式下的重置脚本校验。

5. `MySQL迁移与启用.md`
6. `后续迭代设计-账号反馈MySQL性能.md`
7. `版本记录.md`

部署注意：

1. 服务器执行 `git pull`。
2. 本次没有数据库 schema 变更，不需要执行 `sudo mysql < schema/mysql.sql`。
3. 如果要给旧账号重置密码，必须先执行 `pm2 stop shared-reading-api`，再运行脚本，最后重启后端。
